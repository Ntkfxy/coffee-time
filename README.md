# Coffee Time 🏠☕

เว็บไซต์ร้านกาแฟ "Coffee Time" เป็นโปรเจกต์เว็บแบบง่าย (HTML, CSS, JavaScript) ที่รองรับฟีเจอร์พื้นฐานของร้านกาแฟ เช่น เมนู, ตะกร้า, ระบบ Login, และเช็คเอาต์

---

## 💡 ฟีเจอร์หลัก

1. **ระบบ Login**  
   - Username: `admin`  
   - Password: `1234`  
   - ตรวจสอบการเข้าสู่ระบบก่อนเข้าหน้าเมนู
   - Logout ลบสถานะการเข้าสู่ระบบ

2. **หน้าเมนู**  
   - แสดงเมนูกาแฟพร้อมรูปภาพ
   - ปุ่ม **เพิ่มลงตะกร้า**  
   - ตะกร้าแสดงรายการสินค้า, จำนวน, และราคารวม

3. **ระบบตะกร้า (Cart)**  
   - เพิ่มสินค้าได้ทีละชิ้น  
   - ลบสินค้าแต่ละรายการ  
   - แสดงจำนวนสินค้าและราคารวมอัตโนมัติ  
   - ใช้ **LocalStorage** เก็บข้อมูลตะกร้า ทำให้รีเฟรชหน้าไม่หาย

4. **Checkout**  
   - ตรวจสอบว่าตะกร้าว่างหรือไม่  
   - สั่งซื้อสำเร็จ → ล้างตะกร้า

5. **หน้า About / Contact**  
   - หน้าแยกสำหรับข้อมูลร้านและติดต่อ

6. **Responsive**  
   - หน้าเว็บรองรับมือถือและแท็บเล็ต

## ⚙️ วิธีใช้งาน

1. เปิด `login.html` ใน browser  
2. Login ด้วย Username: `admin`, Password: `1234`  
3. เลือกสินค้าในหน้าเมนู  
4. สินค้าที่เลือกจะเก็บใน LocalStorage  
5. สามารถลบสินค้าออกจากตะกร้า  
6. กด **Checkout** เพื่อล้างตะกร้าและแสดงข้อความสั่งซื้อสำเร็จ
